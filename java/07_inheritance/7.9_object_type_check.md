## **7.9 객체 타입 확인**

- 객체의 타입을 확인하기 위해 **instanceof** 연산자를 사용함.

```java
boolean result = 객체 instanceof 타입;

if(parent instanceof Child child) {
    // child 변수 사용
}
```

- Java 12부터는 `instanceof` 결과가 `true`일 경우, 강제 타입 변환 없이 변수 사용이 가능함.
