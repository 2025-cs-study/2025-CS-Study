### 8.4 추상 메소드
- 인터페이스는 구현 클래스에서 반드시 구현해야 하는 `public` 추상 메소드를 정의함.
- 추상 메소드는 실행 부분이 없이 메소드의 이름, 반환 타입, 매개변수 리스트만을 포함함.
- 구현 클래스에서 추상 메소드를 오버라이딩할 때, 접근 제한자를 `public`으로 설정해야 하며, 더 낮은 접근 제한자로 설정할 수 없음.
