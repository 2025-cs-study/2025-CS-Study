## 8.1 인터페이스의 역할
- **인터페이스(Interface)**: 두 장치를 연결하는 접속기. 두 객체를 역할을 수행함.
- 인터페이스는 다형성을 구현하는 주요 수단으로, 상속을 사용하는 것보다 인터페이스를 이용하는 방법이 더 일반적임.
