### 8.5 디폴트 메소드
- 인터페이스 내부에도 실행 코드를 포함한 디폴트 메소드를 선언할 수 있음.
- 디폴트 메소드는 인터페이스를 구현하는 클래스에 의해 재정의될 수 있음.

  ```java
  [public] default 리턴타입 메소드명(매개변수, ...) { ... }
  ```
