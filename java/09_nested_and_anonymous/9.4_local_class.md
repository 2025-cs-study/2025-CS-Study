# 9.4 로컬 클래스

## 로컬 클래스란?
메소드 내부에서 선언된 클래스를 의미함.

```java
public class A {
    void method() {
        class B {
        }
    }
}
```

### 특징
- **메소드가 실행될 동안만** 객체를 생성할 수 있음.
- Java 17부터 **정적 필드와 정적 메소드 선언이 가능함**.

```java
void useB() {
    class B {
        int field1 = 1;
        static int field2 = 2; // Java 17부터 허용

        void method1() {
            System.out.println("B-method1 실행");
        }

        static void method2() {
            System.out.println("B-method2 실행");
        }
    }
}
```
