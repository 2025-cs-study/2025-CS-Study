# 9.5 바깥 멤버 접근

## 바깥 클래스의 멤버 접근 제한

| 중첩 클래스 유형 | 사용 가능한 멤버 |
|---------------|----------------|
| 인스턴스 멤버 클래스 | 바깥 클래스의 모든 필드와 메소드 |
| 정적 멤버 클래스 | 바깥 클래스의 정적 필드와 정적 메소드만 가능 |

2. 중첩 클래스에서 바깥 클래스의 인스턴스를 참조하려면 어떻게 해야 하나요?
